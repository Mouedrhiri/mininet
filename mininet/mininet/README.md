# MiniNet  Prototype pédagogique (C&C)

MiniNet est un mini réseau social illustrant une architecture Composants & Connecteurs (C&C) avec traçabilité (ACME, PlantUML) vers le code (Node.js/Express, SQLite, EventBus).

Prérequis: Node.js 18+, pdflatex, Bash (Git Bash sous Windows).

Installation:
    make install

Démarrage:
    make start  (http://localhost:3000)

Démo:
    bash backend/scripts/demo.sh

Rapport:
    make report  -> report/rapport.pdf

Présentation:
    make presentation -> presentation/presentation.pdf
